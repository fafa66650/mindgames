# MANUEL MAÎTRE DU JEU – FAFATRAINING V7

## Comment animer
1. Choisis l’âge, la difficulté, le lieu, le type de jeu et le nombre d’équipes.
2. Génère la session.
3. Donne un lien ou un code à chaque équipe.
4. Garde pour toi les réponses et les solutions.
5. Utilise les QR uniquement quand une mission l’exige.

## Comment connaître les réponses
Les réponses attendues sont dans le mode animateur et dans les fichiers de préparation terrain. Les joueurs, eux, ne voient que leurs missions.

## Logique multi-équipes
Chaque équipe a un parcours différent, mais toutes convergent vers la même finale.

## Intérieur / extérieur
- Intérieur : pas d’épreuves physiques brutes.
- Extérieur : défis dynamiques possibles.
