# MANUEL MAÎTRE DU JEU – FAFATRAINING MINDGAME V6

## 1. Logique générale
- 1 session = 1 scénario + 2 à 8 équipes.
- Chaque équipe a son propre lien.
- Chaque équipe voit uniquement ses missions.
- L’animateur garde les réponses dans la console animateur.

## 2. Comment lancer
1. Ouvre l’app.
2. Onglet **Lancer**.
3. Choisis âge, difficulté, lieu, type de jeu, nombre d’équipes.
4. Clique sur **Générer la session**.
5. Copie les liens d’équipes et envoie un lien par équipe.

## 3. Intérieur / extérieur
- **Intérieur** : pas d’épreuves physiques non préparées.
- **Extérieur** : défis dynamiques autorisés.
- Si tu veux un défi intérieur plus physique, il faut le préparer avant avec affiches, zones ou matériel.

## 4. Comment tu connais les réponses
- Dans **Console animateur**, chaque équipe a ses réponses affichées.
- Dans les missions, la réponse attendue est le code généré du type `SCENARIO + EQUIPE + MISSION`.
- Les QR servent surtout à donner une station / un déclencheur clair.

## 5. Conseils terrain
- 6–10 ans : 45 à 60 minutes.
- 11–17 ans : 60 à 90 minutes.
- 18+ : 75 à 120 minutes.
- Prépare un briefing oral clair avant de lancer les liens.
- Vérifie la batterie des téléphones si les équipes jouent en autonomie.
