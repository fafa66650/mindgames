Guide rapide

Comment lancer

Choisis âge, difficulté, lieu, type de jeu et nombre d’équipes. Clique sur “Générer la session”. L’app crée un briefing, un déroulé, des liens d’équipes et la correction animateur.

Comment les équipes jouent

Chaque équipe reçoit son lien ou entre son code dans l’onglet “Mode équipe”. Elle voit uniquement ses missions et valide ses réponses sans voir celles des autres.

Comment tu connais les réponses

Dans la console animateur, tu as toutes les réponses, les QR, les titres des missions et le déroulé minute par minute. Un guide imprimable est aussi fourni dans le ZIP.

Intérieur / extérieur

Intérieur = pas d’épreuves physiques non préparées. Extérieur = défis dynamiques autorisés. L’app adapte automatiquement les types d’épreuves selon le lieu.
