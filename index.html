Guide d’utilisation

1. Administrateur

Prépare la partie, choisis le jeu, le public, la difficulté, le nombre d’équipes et la durée. Partage ensuite les QR ou liens par équipe.

2. Joueurs

Les équipes arrivent avec leur lien ou leur code. Elles peuvent renseigner le nom d’équipe et les joueurs, puis lancer leur chrono global.

3. Simultané

Plusieurs équipes peuvent jouer en parallèle. Dans cette version GitHub Pages, le suivi session est stocké localement dans le navigateur.

4. Réponses animateur

Le mode animateur affiche les réponses attendues, les indices et le suivi des équipes de la session active.
